# PayFlow Database Optimization

## Project Overview
PayFlow is a leading fintech platform processing millions of transactions daily. This project focuses on moving away from inconsistent, local SQL scripts to a centralized, professional data engineering workflow.

## Steps
+ Data was extracted from [here](https://www.kaggle.com/api/v1/datasets/download/olistbr/brazilian-ecommerce)
+ Database was created on Postgres
+ Data has 9 tables but only five tables were used
+ Created five raw tables in our database
+ created 3 final tables PS. Scripts to this table creation can be found in the sql folder

## Tech Stack
+ Database: PostgreSQL 
+ Language: SQL 
+ Version Control: Git & GitHub 
+ Dataset: Brazilian E-Commerce Public Dataset by Olist
